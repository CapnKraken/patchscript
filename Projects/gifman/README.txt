Gifman the Stupendous

Simple 2D platformer. Make it through ten levels, collecting coins to unlock the goal.
If you don't die, you unlock the secret level :)

Controls:
Left Arrow	- move left
Right Arrow	- move right
Z		- jump/UI select
Up Arrow	- UI Up
Down Arrow	- UI Down
Escape		- pause
F11		- toggle fullscreen

For the level editor controls, see Editor_Instructions.txt
